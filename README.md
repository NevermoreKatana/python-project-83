# Проект "Анализатор страниц"

## Описание
Page Analyzer – это сайт, который анализирует указанные страницы на SEO-пригодность по аналогии с PageSpeed Insights

## Установка 

```sh
make build
```

## Запуск сервера
```sh
make start
```
